Angular starter template es un proyecto que implementa las mejores prácticas en la codificación, construcción y prueba de aplicaciones angulares.

Funciones
      
  • Listo para usar, sistema de compilación tipado estático usando Gulp para trabajar con TypeScript.
    •   
  • Desarrollos de producción y desarrollo.
    •   
  • Ejemplos de pruebas de unidades con Jasmine y Karma, incluida la cobertura de código.
    •   
  • Pruebas de extremo a extremo con Cypress.
    •   
  • Servidor de desarrollo con recarga en vivo.
    •   
  • Gestión de definición de TypeScript utilizando Types.
    •   
  • Basic Service Worker, que implementa "Cache then network strategy".
